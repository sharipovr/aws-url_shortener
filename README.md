# AWS URL Shortener

Serverless URL shortener service built with AWS.

## Goal

Build a simple URL shortener to learn AWS serverless architecture:
- AWS Lambda for business logic
- API Gateway for HTTP endpoints  
- DynamoDB for data storage
- Infrastructure as Code with AWS SAM
